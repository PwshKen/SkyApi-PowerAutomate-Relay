Below are the available webhook constituent event types. Subscribe to one of the following events to receive a message when the event has occurred.

This event type includes changes to user biographical information.

 `  com.blackbaud.education.user.change.v1  ` 

| Event data properties | Description | Data type |
| --- | --- | --- |
| `id` | User record identifier | string |

### Related SKY API endpoint

[GET User by ID](https://developer.sky.blackbaud.com/api#api=school&operation=V1UsersByUser_idGet) – Get the user record by using `id`.
